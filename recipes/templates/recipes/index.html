Submit a recipe!

I really need recipe data! Login here using Facebook, Twitter or OpenID and add a recipe. Please make sure to enter each part of your recipe in the correct box. If you have questions, comments, suggestions, complaints, tips, advice, or you just want to say hello, e-mail me at kgodey@kritigodey.com. Bug reports are especially helpful.

About this project

I am a computer science student at Oberlin College and this is the website for my honours project. I am working on recommendation systems for recipes/meal plans based on user preferences, communal ratings, nutritional data and (eventually) food costs. This website is pretty barebones right now, but eventually I hope that it will become actually useful. I need recipes (and ratings) with which to test various recommendation algorithms, and the current purpose of this website is to collect that data.

Planned features

  1. Editing and deleting recipes will actually work!
  2. Search
  3. More polished user accounts, which keep track of the ingredients you have.
  4. Nutritional information for all ingredients.
  5. A system for recipe reviews and comments.
  6. The ability to upload pictures for recipes.
  7. A tagging and categorising system.
