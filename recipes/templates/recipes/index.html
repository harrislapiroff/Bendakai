Submit a Recipe!

Login here using Facebook, Twitter or OpenID and add a recipe.. I'm currently collecting recipes to use as data for my recommendations. Please enter the recipes correctly in the provided fields (and please suggest improvements.) Also note that by submitting recipes and not marking them "private", you grant me an worldwide non-exclusive right to use them as I please. (I plan to release an XML dataset of these recipes to other people that might want to do recipe-based research.)

About the Project

This is Kriti Godey's honours project. It is a web application that will store recipes and help plan meals by recommending recipes based on user preference, general rating and (eventually) nutrition and price.

Upcoming Features

  • Associating your accounts so you can login from multiple ones.
  • A system for reviewing recipes and ingredients
  • The ability to upload pictures for recipes
